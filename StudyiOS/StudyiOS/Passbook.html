#import <PassSlot/PassSlot.h>


// 填写Passslot的APPID

[PassSlot start:@"xxxxxxxxxxxxxxxxxxxxxxx"];


// 定制你的Pass

NSDictionary *values = @{@"label": @"这是一个", @"value": @"测试程序"};

    

// 使用模板创建一个新的PASS

[PassSlot createPassFromTemplateWithName:@"coupon" withValues:values andRequestInstallation:self completion:^{}];


PS:由于Passbook的创建和操作主要在服务器端,比较复杂,这里使用了一个第三方库PassSlot以简化对Passbook的操作

关于PassSlot请看https://www.passslot.com/

想对Passbook进行更深入研究的请看http://www.devdiv.com/iOS_iPhone-iOS6%E6%96%B0%E7%89%B9%E5%BE%81%EF%BC%9A%E5%88%9D%E8%AF%86PassKit-thread-129214-1-1.html及苹果官方API
